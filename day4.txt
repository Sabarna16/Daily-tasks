APTITUDE (Time & Work – Moderate)

Question 1
A can complete a work in 10 days and B in 15 days. How long will they take together?

Solution:
A=10days,B=15days
1/10+1/15=3+2/30=5/30
5/30=1/6=6days

Question 2
A, B and C can complete a work in 6, 8 and 12 days respectively.
How many days will they take working together?

Solution:
A=6days,B=8days,C=12days
LCM=24
4+3+2/24=9/24=3/8
8/3=2.67days

Question 3
A alone can do a work in 12 days. After 4 days, B joins and completes remaining work in 4 days.
Find time taken by B alone.

Solution:
A alone can do a work in 12 days
A in 4days
4*1/12=4/12=1/3
1-1/3=2/3
A+B=2/3/4=2/12=1/6
B=1/6-1/12=1/12
B=12days

Question 4
If 5 workers can complete a work in 8 days, how many workers are required to complete it in 4 days?

Solution:
5 workers can complete a work in 8 days
5*8=40
W=40/4=10
=10days

Question 5
A pipe fills a tank in 6 hours and another empties it in 8 hours.
How long to fill the tank?

Solution:
Filling pipe = 6hours
Empty = 8hours
1/6-1/8=1/24=24hours

Arrays

Today I am going to explain about arrays.

An array is a way to store many values in one variable. All values in an array are of the same type.
We can access each value using a number called an index. The index starts from 0.
real-life example. Think about a row of chairs in a classroom. Each chair has a position number. 
If you want to find a student, you can use the position number. In the same way, in an array, 
we find elements using index numbers.
For example, if we have student marks like 80, 75, 90, and 85, we can store all these marks in one array. 
If I want the first mark, I use index 0.
